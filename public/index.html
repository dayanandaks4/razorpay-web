Setup Guide

Follow these steps to make real payments

1

Install dependencies

Clone/download the project folder, then install packages:

npm install
2

Get Razorpay API Keys

Sign up at Razorpay → Dashboard → Settings → API Keys → Generate Test Key. Copy your Key ID and Secret.

→ Go to razorpay.com
3

Add keys to .env file

Open the .env file in the project and replace the placeholder keys:

RAZORPAY_KEY_ID=rzp_test_YOUR_KEY_HERE
RAZORPAY_KEY_SECRET=YOUR_SECRET_HERE
4

Run the server

Start the app locally:

npm start

# For auto-reload during development:
npm run dev
5

Deploy to Railway (Free Hosting)

Push your project to GitHub, then connect it to Railway for free hosting. Set your .env variables in Railway's dashboard.

git init
git add .
git commit -m "razorpay app"
git push origin main

# Then connect at railway.app
→ Go to railway.app
6

Expiry: Any future date
CVV: Any 3 digits
OTP: 1234

UPI: success@razorpay (test UPI ID)
